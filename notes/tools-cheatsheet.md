# 🛠️ Tools Cheatsheet

## Burp Suite
- Proxy listener: `127.0.0.1:8080`
- Intercept: On/Off toggle in Proxy tab
- Repeater: Ctrl+R to send request
- Forward request: Click Forward button

## Docker
- Start Juice Shop: `docker run -d -p 3000:3000 bkimminich/juice-shop`
- Check running: `docker ps`
- Stop container: `docker stop <id>`

## Kali Linux
- Start Burp: Applications → Web Application Analysis → Burp Suite
- Check IP: `ip a`
